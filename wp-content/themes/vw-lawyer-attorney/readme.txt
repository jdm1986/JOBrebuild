/* Steps to Setup Theme */
/*-----------------------------------------------------------------------------------*/

Step to setup theme static page::

Step 1. Add new page named as "home page" and select the template "Custom Home Page".
Step 2. Go to customizer --> static front page --> check Static page, then select the page which you have added example "home page".

For SLider::

Step 1. Add new page, add title, content and featured image and then publish it.
Step 2. Go to customizer --> VW Settings --> Slider settings --> here you can select the page which you have add for slider.

For Contact Us::

Step 1. Go to customizer --> VW Settings --> Contact us --> here you can add your contat details.

For About us ::

Step 1. Add new post, add title, content and then publish it.
Step 2. Go to customizer --> VW Settings --> About Section --> here you can select the post which you have add for about us.

For Why choose us ::

Step 1. Add new category, then add the post and assign that category to the post, add title, content and featured image and publish it.
Step 2. Go to customizer --> VW Settings --> Why Choose Us --> here you can select the category which you have added.

For General Settings::

Step 1. Go to customizer --> VW Settings --> General Settings --> here you can change the layout of the blog post.
